

XP Styles Manifest

Step 1. Compile the resource script.
Step 2. Add the modXpStyles.bas and YourAppName.res file to the project.
Step 3. Add the InitXpStyles to the main form or module.

NOTE 1: Xp Styles do not run in the VB IDE.

NOTE 2: If you create a manifest the file size must be a multple of 8 bytes.  
The file can be padded with spaces and newlines.
